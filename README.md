# 🧮 EULER fx-Natural

**EULER fx-Natural** is a web-based scientific calculator featuring a high-end classic calculator interface. It combines a retro aesthetic with a modern token-based processing logic for a smooth and accurate user experience.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![JS](https://img.shields.io/badge/JavaScript-ES6+-yellow.svg)
![CSS](https://img.shields.io/badge/Design-Glassmorphism-purple.svg)

## ✨ Key Features

* **Natural Display:** LCD-style screen with mode indicators (Deg/Rad) and status (Shift).
* **Premium Design:** Dark interface with gradients, tactile buttons with depth, and an integrated decorative solar panel.
* **Scientific Functions:** * Trigonometry (sin, cos, tan).
    * Logarithms (log, ln).
    * Power, square roots, and exponents ($x^2, x^3, x^y, \sqrt{x}$).
* **Token-Based Logic:** A system that allows deleting entire functions as a block and handles implicit multiplications.
* **Keyboard Support:** Fully functional using your computer's physical keyboard.
* **Modes:** Quick toggle between Degrees (D) and Radians (R).

## 🚀 Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, Grid, gradients, and interactive animations.
* **Vanilla JavaScript:** Calculation logic, DOM manipulation, and keyboard event handling.
